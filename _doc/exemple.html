Guide d’utilisation des composants UI

Cette page montre comment utiliser les classes définies dans public/css/layout.css avec des exemples et le code correspondant.

Boutons
Se déconnecter 
<button class="btn">Bouton (neutre)</button>
<button class="btn-primary">Action primaire</button>
<button class="btn-secondary">Secondaire</button>
<button class="btn-success">Valider</button>
<button class="btn-warning">Attention</button>
<button class="btn-danger">Supprimer</button>
<a class="disconnect" href="#">Se déconnecter</a>

Tous les boutons partagent la même hauteur.

Messages d’état
Votre action a été effectuée avec succès.

Attention, vérifiez les informations saisies.

<div class="warning">Attention, vérifiez les informations saisies.</div>

Les classes .success et .warning sont prévues pour des messages d’information.

<section class="container">
    <div class="card">
        <div style="padding:16px; border-bottom:1px solid var(--border); background:#fafafa; display:flex; align-items:center; justify-content:space-between; gap:12px;">
            <h1 style="margin:0;">Liste des éléments</h1>
            <a class="btn-primary" href="<?= url_to('route-ajout') ?>">Ajouter</a>
        </div>
        <div class="table-wrap" style="padding: 8px 12px 16px;">
            <?php
            $table = new \CodeIgniter\View\Table();
            // Optionnel: personnaliser l'ouverture de la table (id, classes...)
            $table->setTemplate([
                'table_open' => '<table id="demoListTable">'
            ]);

            $table->setHeading('Nom', 'Prénom', 'Actions');

            foreach ($liste as $row) {
                $table->addRow(
                    esc($row['nom']),
                    esc($row['prenom']),
                    '<a class="btn-warning" href="' . url_to('route-modif', $row['id']) . '">Modifier</a> '
                    . '<a class="btn btn-danger" href="' . url_to('route-suppr', $row['id']) . '">Supprimer</a>'
                );
            }

            echo $table->generate();
            ?>
        </div>
    </div>
</section>

Le code ci-dessus utilise le générateur de table CodeIgniter (\CodeIgniter\View\Table) et produit un HTML stylé automatiquement par votre fichier layout.css.
